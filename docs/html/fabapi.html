fabapi.get_url(filters: Dict[str, List[str]], resource: str, page: int = 0, verbose: bool = False) → str[source]¶

Converts a set of search options to an API URL to be sent to the microbes.caltech.gps.edu server.

Parameters:

  • cols – The names of the columns in the database to access.

  • filters – The filters to apply to each column, given as a tuple of (operator, value), or a single value (equality operator implied).

  • resource – The resource table to access (one of ‘sequences’, ‘metadata’, or ‘annotations’)

  • page – The page of results to retrieve.

Returns:

The URL for accessing the Find-A-Bug web application.
